Play 50 Hz

Play50hz est une application de retro gaming, elle est composée d'un manager, pour gérer votre collection de jeux retro, et d'un player pour les afficher et y jouer. Le manager est votre serveur, c'est lui qui contiendra et gérera toutes voes roms, couvertures, videos et meta-données, que vous partagerez avec les players.

Avec un seul manager, vous pouvez ainsi regrouper toute votre collection, et la partager facilement sur toutes vos machines.

Téléchargement et installation

Vous pouvez télécharger play50hz ici.

Une fois le téléchargement terminé, décompressez l'archive ZIP, dans le dossier vous trouverez le manager, vous aurez besoin de java version 11 minimum pour le faire tourner.

Le manager fonctionne sous windows ou linux.

Dans un un terminal (Touche Windows + x, puis i sous windows, CTRL + ALT + t sous linux), naviguez jusqu'au dossier où se trouve le manager, en utilisant la commande 'cd', et enfin entrez la commande pour lancer le manager: ' java - jar retro-manager-assembly.jar '

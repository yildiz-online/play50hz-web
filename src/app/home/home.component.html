Play 50 Hz
Un seul programme pour tout gérer

Play50hz est une application de retro gaming, elle est composée d'un serveur(le manager), pour gérer votre collection de jeux retro, et d'un front-end(le player) pour les afficher et y jouer.

C'est le manager qui contiendra et gérera toutes voes roms, couvertures, videos et meta-données, que vous partagerez avec les players qui s'y connecteront.

Reconnaissance intelligente des jeux

Déposez simplement vos roms dans le dossier que vous aurez choisi, faite de même pour vos covers, vos videos, le programme se charge de tout réconcillier intelligemment, même si les noms de fichiers ne correspondent pas.

Moins de temps à gérer, plus de temps pour jouer

Avec la gestion centralisée de vos jeux, données, images et vidéos, qui seront partagés entre toutes vos machines, vous ne passerez presque plus de temps à configurer les systèmes, à télécharger, à ranger, dès que vous changez de machine, tout est centralisé et partagé.

Profitez du temps gagné pour enfin atteindre les 101% sur Donkey Kong Country!

Téléchargement et installation

Vous pouvez télécharger play50hz ici.

Une fois le téléchargement terminé, décompressez l'archive ZIP, dans le dossier vous trouverez le manager, vous aurez besoin de java version 11 minimum pour le faire tourner.

Le manager fonctionne sous windows ou linux.

Dans un un terminal (Touche Windows + x, puis i sous windows, CTRL + ALT + t sous linux), naviguez jusqu'au dossier où se trouve le manager, en utilisant la commande 'cd', et enfin entrez la commande pour lancer le manager: ' java - jar retro-manager-assembly.jar '

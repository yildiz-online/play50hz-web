Un seul programme pour tout gérer

Play50hz est une application de retro gaming, elle est composée d'un serveur(le manager), pour gérer votre collection de jeux retro, et d'un front-end(le player) pour les afficher et y jouer.

C'est le manager qui contiendra et gérera toutes vos roms, couvertures, videos et meta-données, que vous partagerez avec les players qui s'y connecteront.

Reconnaissance intelligente des jeux

Déposez simplement vos roms dans le dossier que vous aurez choisi, faite de même pour vos covers, vos videos, le programme se charge de tout réconcillier intelligemment, même si les noms de fichiers ne correspondent pas.

Moins de temps à gérer, plus de temps pour jouer

Avec la gestion centralisée de vos jeux, données, images et vidéos, partagés entre toutes vos machines, vous ne passerez presque plus de temps à configurer les systèmes, à télécharger, à ranger. Dès que vous changez de machine, tout se téléchargera depuis votre manager.

Profitez du temps gagné pour enfin atteindre les 101% sur Donkey Kong Country!

Téléchargement et installation

Vous pouvez télécharger play50hz ici .

Une fois le téléchargement terminé, décompressez l'archive ZIP (sous windows,clic droit extraire tout, attention à donc bien extraire et non pas lancer le programme depuis l'archive). Dans le dossier extrait, vous trouverez le manager, vous aurez besoin de java version 11 minimum installé sur votre machine pour le faire tourner(Version windows téléchargeable ici.

Le manager fonctionne sous windows ou linux.

La dernière étape avant de lancer le manager est d'éditer le fichier de configuration(configuration.properties) :

  • system.support Les systèmes supportés, actuellement seuls snes(super nintendo), nes (nintendo), sms (master system) et gen (megadrive) sont supportés.
  • video.directory Le dossier où se trouveront les vidéos de vos jeux, elles devront être dans un sous dossier nommé comme le système (snes, nes, gen, sms).
  • cover.directory Le dossier où mettre vos covers de jeu, aussi dans un sous dossier avec le nom du système.
  • roms.directory Le dossier où déposer vos roms, toujours dans un sous dossier système.
  • player.directory Le dossier où se trouve 'player.zip', par défaut ce devrait être le dossier du manager/player.
  • csv.directory Le dossier où sera traitée la base de donnée CSV.
  • web-server.port Le port de communication entre le manager et les players.
  • theme.directory Le dossier où se trouvent les fichiers de thème du manager, par défaut dossier du manager/themes
  • csv.account Le dossier où sont stockés les fichiers de comptes utilisateurs.
  • launchbox.directory Le dossier où sera téléchargée la base de donnée launchbox.
  • clones.directory Le dossier où se trouvent les listes de clones(gen-clones.txt, snes-clones.txt,...)
  • clean.directory LE dossier où se trouvent les listes de nettoyage de nom(gen-clean.txt, snes-clean.txt...)

Sous windows, vous pouvez éxecuter le script de démarrage "run.bat", sous linux, dans un terminal, rendez vous dans le bon dossier et lancez la commande ' java - jar retro-manager.jar '

Une fois le démarrage terminé, sur la machine où vous voulez jouer, ouvrez votre navigateur et entrez l'url suivante: http://'ip de la machine du manager':8080/setup
